Valid Usernames:
- user123
- alice
- bob_123
- test-user
- a
- _user
- -test

Invalid Usernames:

- `` (empty)
- user name (contains space)
- user@host (contains @)
- user! (contains !)
- user# (contains #)
- user. (contains .)
- user* (contains *)

Test cases:
- wrong password
- wrong nickname
- wrong user name
- wrong privmsg
